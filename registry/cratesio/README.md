# ltm-cli

**This crate is a name reservation.** The real ltm CLI is a Go binary — install it from https://ltm-cli.dev or https://github.com/dennisdevulder/ltm.

A Rust SDK for the [Core Memory Packet protocol](https://github.com/dennisdevulder/ltm/blob/main/SPEC.md) may ship under this name in the future.
