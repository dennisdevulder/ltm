# ltm-cli

**This package is a name reservation.** The real ltm CLI is a Go binary — install it from https://ltm-cli.dev or https://github.com/dennisdevulder/ltm.

A Python SDK for the [Core Memory Packet protocol](https://github.com/dennisdevulder/ltm/blob/main/SPEC.md) may ship under this name in the future.

## What is ltm?

`ltm` moves the *intent and state* of a work session between machines, models, and agents — without dragging along your configuration.

- Spec: https://github.com/dennisdevulder/ltm/blob/main/SPEC.md
- Homepage: https://ltm-cli.dev
- Repository: https://github.com/dennisdevulder/ltm
